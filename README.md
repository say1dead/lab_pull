def area(r):
    return math.pi * r * r

def perimeter(r):
    return 2 * math.pi * r  
В функцию area(r) мы передаем значение r - радиус круга -> получаем его площадь  
В функцию perimetr(r) мы передаем значение r - радиус круга -> получаем длину его окружности
